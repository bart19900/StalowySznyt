# Aktualizacja modeli 3D

Skopiuj zawartość pakietu do głównego folderu repozytorium i zastąp stare pliki.

Oczekiwane ścieżki:

```text
models/lampa-kwietnik/lampa-kwietnik.glb
models/stolik-kawowy/stolik-kawowy.glb
models/kwietnik-narozny/kwietnik-narozny.glb
models/kwietnik-pionowy-4-pietrowy/kwietnik-pionowy-4-pietrowy.glb
```

Jeżeli nazwy są inne, zmień pole `model` w odpowiednim `product.json`.
Następnie uruchom `build-gallery.bat`, zrób Commit to main i Push origin.
